# /dashboard

Displays a progress dashboard for migrations and large projects.

## What it does
Generates a visual summary of project status: modules migrated, tests passing, coverage, pending work.

## Instructions
1. Read migration plan from `.cloud/planning/migration-plan.md` (if exists)
2. Scan test results and coverage
3. Check audit trail for completed modules
4. Generate a markdown dashboard with progress bars and status indicators
5. Display the dashboard to the user

## Usage
```
/dashboard
```
