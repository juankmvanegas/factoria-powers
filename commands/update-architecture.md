# /update-architecture

Updates architecture documentation after structural changes.

## What it does
Synchronizes `.cloud/architecture/` docs with the current state of the codebase after significant changes.

## Instructions
1. Analyze current project structure
2. Compare against existing architecture docs in `.cloud/architecture/`
3. Update `current.md` with any structural changes
4. Update ADR references if new decisions were made
5. Update `CLAUDE.md` if conventions or stack changed

## Usage
```
/update-architecture
```
