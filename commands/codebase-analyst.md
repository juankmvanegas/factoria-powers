# /codebase-analyst

Analyzes existing code to understand structure, patterns, and quality.

## What it does
Deep-reads the codebase and generates a structured analysis report covering architecture, patterns, dependencies, and potential issues.

## Instructions
1. Scan the project structure layer by layer
2. Identify all entities, services, repositories, and their relationships
3. Check adherence to architecture rules (4-layer Clean Architecture)
4. Identify patterns used and any deviations from standards
5. Report findings with recommendations

## Usage
```
/codebase-analyst [optional: specific area to analyze]
```
