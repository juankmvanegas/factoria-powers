# /update-factory

Updates the Factoria framework files (CLAUDE.md, skills, commands, policies).

## What it does
Modifies Factoria's own configuration and documentation to reflect new learnings, standards, or improvements.

## Instructions
1. Identify what needs to be updated (skill, policy, CLAUDE.md section, ADR table)
2. Make the changes following existing formatting conventions
3. Verify consistency across all related files
4. Report what was changed

## Usage
```
/update-factory [what to update]
```
