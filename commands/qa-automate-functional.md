# /qa-automate-functional

Implements or updates functional QA automation from approved cases.

## What it does
Adds or updates automated coverage in the appropriate testing layer and records traceability back to the QA artifacts.

## Instructions
1. Read the automation plan and traceability matrix
2. Implement or update automated coverage for the selected cases
3. Follow `.cloud/policies/testing-policy.md`
4. Update the traceability matrix with automation references

## Usage
```text
/qa-automate-functional [test case ids or feature]
```
