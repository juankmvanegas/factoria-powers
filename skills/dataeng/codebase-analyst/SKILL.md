# Skill: Codebase Analyst

## Purpose

Analyze an existing Data Engineering codebase to identify data pipeline patterns, pipeline reproducibility, artifact management, testing gaps, and technical debt.

## What to inspect

- `main.py`
- `databricks.yml`, `Databricks job run metadata`, `params.yaml`
- `databricks.yml`
- runtime artifact expectations
- notebooks and extracted analytical helpers
- monitoring and operational assets
