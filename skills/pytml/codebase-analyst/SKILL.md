# Skill: Codebase Analyst

## Purpose

Analyze an existing Python MLOps codebase to identify serving patterns, pipeline reproducibility, artifact management, testing gaps, and technical debt.

## What to inspect

- `main.py`
- `dvc.yaml`, `dvc.lock`, `params.yaml`
- `databricks.yml`
- runtime artifact expectations
- notebooks and extracted analytical helpers
- monitoring and operational assets
